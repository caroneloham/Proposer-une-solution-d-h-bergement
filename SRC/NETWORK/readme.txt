# Toolkit de Gestion de Réseau et Base de Données

## Description
Ce package de scripts Python offre une série d'outils pour la gestion de projets de configuration de réseaux et de bases de données. Il comprend une visualisation de la progression des tâches via une Road Map, la configuration de DNS, et des opérations sur bases de données SQL.

## Fonctionnalités
- **Road Map de Projets**: Génère un diagramme de Gantt pour visualiser le calendrier des différentes tâches liées à la configuration et à la sécurisation des réseaux informatiques.
- **Configuration DNS**: Permet de configurer et de tester des settings DNS via un script automatisé.
- **Gestion de Base de Données SQL**: Exécute des requêtes SQL et gère les bases de données pour optimiser les interactions et la sécurité.

## Installation
### Prérequis
- Python 3.x
- Bibliothèques Python : Matplotlib, datetime, sqlite3 (ou une autre librairie de gestion SQL selon votre base de données)
- Pour les tâches spécifiques liées à DNS et SQL, des configurations supplémentaires peuvent être nécessaires selon votre environnement.
