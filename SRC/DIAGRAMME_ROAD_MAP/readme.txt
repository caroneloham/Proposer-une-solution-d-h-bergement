# Road Map pour Configuration et Sécurité Réseau

## Description
Ce script Python utilise la bibliothèque Matplotlib pour générer une visualisation sous forme de Road Map des différentes étapes d'un projet de configuration et de sécurisation de réseaux informatiques. Le script affiche un diagramme avec les dates de début et de fin pour chaque tâche, mettant en évidence les différentes phases du projet par des couleurs spécifiques.

## Fonctionnalités
- Visualisation des tâches avec leur durée et dates de début sur un diagramme de Gantt.
- Différenciation visuelle des types de tâches par des couleurs distinctes.
- Affichage des noms des membres de l'équipe associés à chaque phase.
